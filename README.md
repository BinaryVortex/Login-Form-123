# Login-Form-123

![Login Form Screenshot](https://github.com/BinaryVortex/Login-Form-123/blob/main/Screenshot%202025-02-05%20123746.png)

A simple, responsive login form built with HTML, CSS and JavaScript.

## Features

- Clean, modern UI built with semantic HTML and CSS.
- Client-side validation using JavaScript (username/password checks).
- Responsive layout that works on mobile and desktop.

## Demo

To view the form locally:

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-123.git

2. Open `index.html` in your browser (double-click or use `File > Open`).

No build tools or server required.

## Files

- `index.html` — main markup for the login form.
- `styles.css` — styling and responsive layout.
- `script.js` — JavaScript for client-side validation and interactions.
- `Screenshot 2025-02-05 123746.png` — screenshot used above.

## Usage

- Customize the HTML/CSS to match your project's branding.
- Enhance validation or integrate with a backend authentication endpoint as needed.
